build notes

Choices and small problems

  • Kept the class data synthetic and labeled clearly so it does not read like real medical or genetic research.
  • Used MICE for missing values instead of throwing away useful rows too quickly.
  • Kept the selected transformations from the original work, then made the outputs easier to rerun and inspect.
  • Saved summaries, ANOVA tables, coefficients, and plots as separate project outputs.
